# Troll Website

A fun and harmless troll website built with simple HTML, CSS, and JavaScript. This project is designed for playful experiments and lighthearted trolls. It's easy to run and requires no build tools.

---

## Features

- **Interactive trolls**: Includes fake loading screens, runaway buttons, and humorous pop-ups.
- **Lightweight**: No dependencies or frameworks required.
- **Customizable**: Easily tweak the styles and scripts to create your own trolls.

---

## Project Structure

- **`index.html`**: The main entry point of the troll website.
- **`style.css`**: Contains all the styles for the troll elements.
- **`script.js`**: Implements the troll logic and interactivity.

---

## How to Run

This is a static website, so running it is simple:

### Option 1: Open Directly
1. Locate the `index.html` file in the project folder.
2. Open it in your browser (double-click or right-click â†’ Open with...).

### Option 2: Run a Local Server (Recommended)
If the troll uses modules or fetch requests, running a local server is recommended:

1. Open a terminal in the project directory.
2. Run the following command to start a simple HTTP server:
   ```bash
   python3 -m http.server
   ```
3. Open `http://localhost:8000` in your browser.
